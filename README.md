Menstruation Tracker

A simple Python program that allows users to track their menstrual cycle, record dates, and estimate the next period date. Designed for beginners, the project focuses on basic Python concepts such as input handling, date calculations, and simple data storage.


---

Features:

Record the start date of the last period

Enter average cycle length

Estimate the next expected period

Display cycle information clearly

Beginner-friendly logic and structure



---

How It Works

1. The user enters the date of their last period.


2. The program takes their average cycle length in days.


3. It calculates the estimated next period date.


4. It displays the results in a simple, readable format.






Requirements

Python 3

Uses only built-in modules (datetime)





Usage

Run the script and follow the on-screen instructions to enter your last period date and cycle length. The program will calculate and display your next expected cycle date.




Target Users

Individuals who want a simple, offline menstrual cycle tracker

Students learning Python basics

Beginners practicing date and input handling





Scope

The project focuses on simple tracking and date prediction. It does not include notifications, data storage, or health analytics. These can be added in future versions.
