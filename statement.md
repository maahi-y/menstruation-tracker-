Problem Statement

Users often struggle to track their menstrual cycle manually, which can lead to missed dates, irregular pattern awareness, and difficulty predicting future cycles. A simple digital tracker can help maintain accurate records and reminders.

Scope of the Project

The project provides a basic console-based menstruation tracker where users input their last period date and average cycle length. The system calculates the estimated next period date and fertile window. It focuses on accuracy, simplicity, and beginner-friendly implementation.

Target Users

Individuals who want a minimal, easy-to-use tool to track their menstrual cycle without complex apps or external dependencies.

High-Level Features

Input last period date and cycle length

Predict next period date

Estimate fertile window

Store and display basic cycle information

Beginner-friendly, module-free Python implementation
